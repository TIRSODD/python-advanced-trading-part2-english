# Python for Advanced Algorithmic Trading - Part 2
## Volatility, Risk, and Portfolios

Official code repository for the book **"Python for Advanced Algorithmic Trading - Part 2: Volatility, Risk, and Portfolios"**. 

This repository contains the complete, production-ready Python code developed throughout the book. It covers advanced volatility modeling, dynamic risk management, robust backtesting techniques to avoid overfitting, and the construction of multi-asset, multi-strategy portfolios.

---

##  Repository Structure

The code is organized by chapters, matching the structure of the book:

- **`chapter_01_volatility_modeling/`**: Tools for measuring and modeling volatility (ATR, GARCH models, Market Regime Detection).
- **`chapter_02_volatility_strategies/`**: Adaptive trading strategies based on volatility regimes (Mean Reversion, Momentum, Volatility Squeeze).
- **`chapter_03_dynamic_risk/`**: Dynamic position sizing, trailing stops, and survival filters (Circuit Breakers).
- **`chapter_04_robust_backtesting/`**: Anti-overfitting techniques (Walk-Forward Analysis, Monte Carlo Simulations, Statistical Significance Tests).
- **`chapter_05_eurgbp_case/`**: Case Study: Building a complete multi-strategy system for EUR/GBP.
- **`chapter_06_metals_case/`**: Case Study: Adapting systems for Gold (XAU/USD) and Silver (XAG/USD).
- **`chapter_07_portfolio/`**: Portfolio construction, capital allocation methods (Risk Parity, Kelly Criterion), and correlation analysis.
- **`utils/`**: Helper functions and shared utilities.
